# Open Graph Image Specifications

## Image Requirements

**File:** `og-image.png`  
**Location:** Root directory (`/og-image.png`)  
**URL:** `https://curatedai.net/og-image.png`

### Technical Specs
- **Dimensions:** 1200 x 630 pixels (1.91:1 ratio)
- **Format:** PNG (with transparency support) or JPG
- **File size:** Under 1MB (optimize for web)
- **Color space:** RGB

### Design Guidelines

**Content to include:**
1. **Logo/Brand:** "AI Tool Directory" or site logo
2. **Tagline:** "200+ Curated GenAI Tools" or similar
3. **Visual elements:** 
   - Terminal/tech aesthetic (matches site theme)
   - Subtle grid or code-like elements
   - Modern, clean design
4. **Colors:** Match site theme (dark/light compatible)

### Design Tools
- **Figma** - Create design, export as PNG
- **Canva** - Use "Open Graph Image" template (1200x630)
- **Photoshop** - Design and export optimized PNG
- **AI Tools** - Use Midjourney/DALL-E with prompt: "Professional Open Graph image for AI tools directory, 1200x630, terminal theme, modern, clean"

### Example Prompt for AI Image Generation

**For Midjourney/DALL-E:**
```
Professional Open Graph social media image, 1200x630 pixels, terminal/tech aesthetic, dark charcoal background (#0a0a0a) with subtle monospace grid pattern, minimalist design, text "AI Tool Directory" in monospace font, tagline "Handpicked, not submitted" below, clean typography, terminal theme, no emojis, no icons, just text and subtle grid, professional, high quality, 8k, sharp focus
```

**Alternative (lighter version):**
```
Open Graph image 1200x630, terminal theme, light warm background (#fafaf9), subtle dot grid pattern, monospace typography, text "AI Tool Directory" and "Handpicked GenAI Tools", minimalist, clean, Cursor directory inspired, professional, high quality
```

**Key design elements:**
- Terminal/monospace aesthetic (matches site theme)
- Subtle grid or dot pattern (not overwhelming)
- Minimal text: "AI Tool Directory" + "Handpicked, not submitted"
- Dark or light background (match site theme)
- No emojis, no icons, no illustrations
- Clean, professional, modern

### Testing
After creating the image:
1. Upload to root directory

   - [Twitter Card Validator](https://cards-dev.twitter.com/validator)
